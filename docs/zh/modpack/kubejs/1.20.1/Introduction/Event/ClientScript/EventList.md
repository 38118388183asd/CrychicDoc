# 客户端脚本事件列表

## 前言

- 放在client_script文件夹下生效。

## 事件列表

### ClientEvents 客户端事件

|   事件名称    |   用途    |   示例    |
|:------------:|:---------:|:---------:|
|   atlasSpriteRegistry   |   -   |   -   |
|   highPriorityAssets   |   高优先级资源包加载时触发。   |   -   |
|   lang   |   语言文件加载时触发。   |   -   |
|   leftDebugInfo   |   左侧调试栏信息。   |   -   |
|   loggedIn   |   登录时触发。   |   -   |
|   loggedOut   |   登出时触发。   |   -   |
|   paintScreen   |   渲染paint时触发。   |   -   |
|   painterUpdated   |   paint更新时触发。   |   -   |
|   rightDebugInfo   |   右侧调试栏。   |   -   |
|   tick   |   每刻触发   |   -   |
